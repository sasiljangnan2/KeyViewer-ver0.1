# ZCJ_KeyViewer - ZCJ 키뷰어
> **key** [kiː]  
> _명사_
>
> 1\. 자물쇠를 잠그거나 여는 데 사용하는 물건.<br>
> 2\. 어떤 문제를 해결할 수 있는 실마리.<br>
> 3\. 컴퓨터 키보드에서 손가락으로 눌러 문자, 숫자, 기호 등을 입력하는 부품의 집합체.<br>

> **viewer** [ˈvjuːə(r)]  
> _명사_
>
> 1\. 텔레비전을 비롯하여, 무언가를 시청하는 사람.<br>
> 2\. (IT) 이미지를 볼 수 있게 하는 장비 또는 소프트웨어.<br>
## 소개
ZCJ_KeyViewer(ZCJ 키뷰어)는 **OBS 방송에 최적화된 실시간 키보드 입력 시각화 도구**입니다.
# 사용 방법
## 실행
<ins>keyviewer.exe</ins> 파일을 실행하시기 바랍니다.
## 키 패널 설정
실행된 프로그램에서 마우스 우클릭을 하여 접근 가능한 도구 상자에서 **키 패널 추가**를 선택하세요.<br>
**단축키 설정** 버튼을 눌러 단축키를 설정할 수 있습니다. 단축키와는 별도로 표기되는 키의 이름을 따로 설정할 수 있습니다.

키 패널의 기본적인 디자인과 위치 변경이 가능합니다. 기본 색상과 눌렸을 시의 색상, 테두리, 채도, 그리고 배열 등을 본인이 원하는 대로 커스터마이징하시기 바랍니다.
## 전역(Global) 설정
키 패널을 하나하나 설정하는게 귀찮거나 그럴 필요성을 느끼지 못하신다면, **전역 설정**을 이용하세요.<br>
화면상의 키 패널의 디자인을 통일할 수 있습니다.

또한 전역 설정에서는 배경 화면의 설정도 가능합니다. 자신이 원하는 색상이나 이미지를 적용해보세요.
## 레이아웃
프로그램을 실행할 때마다 설정할 필요 없이, 기존에 만들어 놓은 상태를 저장할 수 있습니다.<br>
**레이아웃 저장**을 눌러 원하는 배열을 저장하고, **레이아웃**을 눌러 불러올 수 있습니다.
## OBS 호환
OBS 상에 프로그램을 띄우고 싶다면, 도구 상자에서 **OBS 호환 모드**를 활성화하세요.<br>
호환 모드 비활성화 상태에서는 OBS 상에 프로그램이 표시되지 않습니다.<br>
또한 OBS에서는 *윈도우 캡쳐*로 소스를 생성하시기 바랍니다. 타 소스는 캡쳐가 되지 않습니다.

크로마키를 사용하신다면, 전역 설정에서 크로마의 색을 지정한 뒤 **배경 완전 투명화** 버튼을 활성화하시기 바랍니다.<br>
그리고 OBS에 생성해놓은 소스를 마우스 우클릭하여 *필터*로 들어간 뒤, *효과 필터*에서 '크로마 키'를 추가하세요.<br>
키 색상 형식은 전역 설정에서 지정해놓은 크로마 색과 같은 색으로 설정하시면 됩니다.
